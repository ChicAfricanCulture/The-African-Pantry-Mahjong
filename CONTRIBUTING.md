# Contributing to The African Pantry Mahjong

Thank you for your interest in helping grow this project! The African Pantry Mahjong is a free, educational browser game that celebrates African ingredients, foodways, and recipes through mahjong-style matching. We're building it together to make learning about African cuisine fun and accessible.

This project is open source under the MIT License, so feel free to fork, modify, and share. Whether you're a coder, a cook, a food history enthusiast, or just someone who loves African flavors—your input is valuable.

## Ways to Contribute

### No Coding Needed
- **Suggest new ingredients or tiles** — Tell us about staples from your region (e.g., fonio, baobab fruit, moringa, injera ingredients). Include why it's important culturally, any sensory notes (taste, texture), or recipe ideas.  
  Post in Discussions > "Ingredient Ideas" category, or open an Issue.
- **Share recipes or facts** — Link to or describe traditional dishes using the pantry items. We can add pop-up info or links when tiles are clicked.
- **Give feedback on the game** — Play it at https://chicafricanculture.github.io/The-African-Pantry-Mahjong/ and comment: What works? What's confusing? Any bugs? Ideas for new levels (e.g., East African theme, spices-only)?
- **Spread the word** — Share the game on social media, forums, or with friends/family interested in African culture/food.

### Code or Design Contributions
The game is built with plain HTML, CSS, and JavaScript (no frameworks yet). It's client-side only.

1. **Discuss first** — Before writing code, open a Discussion or Issue to talk about your idea (new feature, bug fix, UI tweak, etc.). This avoids duplicate work.
2. **Fork & branch** — Fork the repo, create a branch (e.g., `feature/add-okra-tile` or `fix-mobile-layout`).
3. **Make changes** — Add/edit files thoughtfully. For new tiles: add black-and-white SVG/PNG to `/images/`, update the ingredient list in JS, and add any info text.
4. **Test locally** — Open `index.html` in your browser to check.
5. **Submit a Pull Request** — Describe what you changed and why. Reference any related Issue/Discussion.
6. **Follow basics** — Keep code clean, add comments if needed, match existing style (simple and readable).

### Our Guidelines
- Be respectful and inclusive—everyone's knowledge of African foodways varies by region and experience.
- Focus on positive, accurate representation of cultures and ingredients.
- We follow GitHub's community norms—no spam, hate, etc.

If you're new to GitHub or open source, no worries! Ask questions in Discussions—we're happy to guide you step by step.

Questions? Jump into Discussions or tag @ChicAfricanCulture in an Issue.

Let's keep celebrating African pantry treasures together! 🌿🍲
